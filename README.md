# 🌿 Wildcat's Wonderful Weed List (WWWL)

> A personal cannabis strain directory with 300+ strains, interactive filtering, strain tracking, and a cyberpunk Matrix-green aesthetic — all running as a static GitHub Pages site.

---

## ✨ Features

### 🔍 Live Search
Type any strain name into the search bar and results filter instantly with smooth fade animations. Multi-word searches are supported — all words must match. If no strains are found, an animated "No strains found!" popup appears with a surprise sound effect.

### 🌿 Type Filtering (Legend Bar)
Click the sticky filter bar at the top to narrow by strain type:
- 🔵 **Indica** — Blue cards
- 🔴 **Sativa** — Red cards
- 🟢 **Hybrid** — Green cards
- 🟡 **Unknown** — Yellow cards
- 🌈 **Show All** — Resets all filters
- 🩵 **Tried** — Shows only your marked strains

The legend hides automatically when you scroll down, and reappears when you scroll back up.

### ✅ Tried Strain Tracker
Each strain card has a clickable circle toggle. Mark strains you've tried — they turn teal and bold. Your selections are saved automatically to `localStorage`, so they persist between visits without any account needed.

### 🎲 Random Strain Picker
Hit the **"Pick random?"** button to highlight a random visible strain with a glowing pulsing animation. It scrolls the card into view so you never have to decide what to try next.

### 💾 CSV Export
Click **"Save My Strains"** to download a `.csv` file of all your tried strains. The export includes the strain name, type (Indica/Sativa/Hybrid/Unknown), and a direct Leafly/Allbud link for each one.

### 📊 Live Strain Counts
The filter bar displays live counts next to each category label, updating in real time as you search or toggle tried strains.

### 💬 Community Comments
A [Cusdis](https://cusdis.com)-powered comments section is embedded at the bottom, styled to match the dark Matrix theme. Leave notes, reviews, or suggestions for any strain.

### 🎨 Cyberpunk / Matrix Theme
Built with a dark background, glowing green (`#00ff00`) accents, frosted-glass containers, and smooth CSS animations throughout. Fully responsive for mobile and desktop.

Open index.html in any browser, or visit the live GitHub Pages version:
https://sunnyskyess420.github.io/williamsstrains/

📋 Strain Data
Strains are manually curated and linked to Leafly, Allbud, Weedmaps, and other cannabis databases. Each strain is categorized as Indica, Sativa, Hybrid, or Unknown. The list currently contains 300+ strains and grows over time.
